# Velvet Gifting - E-Commerce Website

A premium, fully functional e-commerce website built with Vite, React, Tailwind CSS, and shadcn/ui.
## Features
- **Modern UI/UX:** Clean, elegant design out of the box.
- **Easy Theming:** Brand colors managed in `src/index.css` via CSS variables.
- **Global Config:** Update brand details easily in `src/config/site.ts`.
- **Zero-Cost Architecture:** Local cart persistence and WhatsApp ordering routing.

## Development
```bash
npm install
npm run dev
```
